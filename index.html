HyperText Markup Language

Text documents containing HTML markup (such documents traditionally have the .html or .htm extension) are processed by special applications that display the document in its formatted form. Such applications, called "browsers" or "Internet browsers", usually provide a user-friendly interface for requesting web pages, viewing them (and displaying them on other external devices) and, if necessary, sending user-entered data to a server. The most popular browsers today are Google Chrome, Mozilla Firefox, Opera, Internet Explorer and Safari.
